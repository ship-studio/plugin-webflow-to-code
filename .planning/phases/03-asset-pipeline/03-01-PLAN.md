    **Step 1: Create `src/assets/types.ts`** with these exact types:
    ```typescript
    export type AssetType = 'image' | 'video' | 'font' | 'svg' | 'animation';

    export interface ImageEntry {
      filename: string;
      path: string;
      type: 'image' | 'svg' | 'animation';
      purpose: string;
      variants?: string[];
      referencingPages: string[];
    }

    export interface VideoEntry {
      filename: string;
      path: string;
      type: 'video';
      purpose: 'video';
      transcodes?: string[];
      poster?: string;
      referencingPages: string[];
    }

    export interface FontEntry {
      filename: string;
      path: string;
      type: 'font';
      purpose: 'font';
      referencingPages: string[];
    }

    export interface AssetManifest {
      images: ImageEntry[];
      videos: VideoEntry[];
      fonts: FontEntry[];
      cssFiles: string[];
      totalCopied: number;
    }
    ```

    **Step 2: Write failing tests in `src/assets/manifest.test.ts`** covering all behaviors listed above. Use the Moneystack sample entries from the context `<interfaces>` block as test fixtures. Import from `../assets/manifest` and `../assets/types`.

    **Step 3: Create `src/assets/manifest.ts`** implementing:
    - `stripVariantSuffix(filename: string): string` — regex `/(-p-\d+(?:x\d+)?(?:q\d+)?)(\.[^.]+)$/` replaces with `$2`
    - `inferImagePurpose(filename: string): string` — priority: favicon.png/webclip.png -> "favicon", contains "logo" -> "logo", contains "placeholder" -> "placeholder", .svg -> "icon-or-graphic", .gif -> "animation", default -> "image"
    - `groupResponsiveVariants(imageEntries: string[], assetsDir: string): ImageEntry[]` — groups by stripped base name, handles variant-only files by promoting the variant filename as canonical, excludes SVGs from variant grouping (SVGs get own entries)
    - `buildVideoGroups(videoEntries: string[], assetsDir: string): VideoEntry[]` — groups source + transcodes + poster by base name prefix
    - `buildManifest(entries: string[], assetsDir: string, projectPath: string): AssetManifest` — filters entries by directory prefix (images/, videos/, fonts/, css/), delegates to grouping functions, builds cssFiles list, computes totalCopied

    Key implementation details:
    - `assetsDir` is absolute (e.g., "/proj/.shipstudio/assets"), but manifest paths must be project-relative. Use `assetsDir.replace(projectPath + '/', '')` to produce paths like ".shipstudio/assets/images/hero.png"
    - Filter out directory entries (ending with "/") before processing
    - For variant-only images (where `stripVariantSuffix` produces a name not in the entries list), use the original variant filename as the entry's filename
    - SVG files in images/ get their own ImageEntry with type "svg", not grouped with raster variants
    - GIF files get type "animation"
    - All referencingPages: [] (Phase 4 populates)

    **Step 4: Run tests, iterate until all green.**
