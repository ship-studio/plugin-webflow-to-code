Create src/components/MigrationProgress.tsx. This component:

**Props interface:**
```typescript
interface MigrationProgressProps {
  shell: { exec(cmd: string, args: string[]): Promise<{ exit_code: number; stdout: string; stderr: string }> };
  projectPath: string;
}
```

**State:**
- `plan: MigrationPlan | null` — parsed plan data
- `pollError: boolean` — true only if file was previously readable and now fails
- `expanded: Set<number>` — indices of expanded page items
- `hadPlan: useRef<boolean>` — tracks whether we ever successfully read the plan

**Polling (useEffect):**
- On mount, call loadMigrationPlan immediately, then setInterval every 30_000ms
- If result is not null: setPlan(result), set hadPlanRef.current = true, setPollError(false)
- If result is null AND hadPlanRef.current is true: setPollError(true) (file was readable before)
- If result is null AND hadPlanRef.current is false: do nothing (file not created yet — silent)
- Cleanup: clearInterval on unmount
- Dependencies: [shell, projectPath]

**Expand/collapse:**
- `toggleExpanded` via useCallback — toggles index in the Set (multiple open allowed, NOT accordion)

**Rendering structure (ALL inline styles, no new CSS classes):**

1. **Section label** — "Migration Progress" text styled to match `.wf2c-results-output-label`:
   ```
   fontSize: '11px', fontWeight: 500, color: 'var(--text-muted)',
   textTransform: 'uppercase', letterSpacing: '0.5px', marginBottom: '8px'
   ```

2. **Error state** — if pollError is true AND plan is null:
   ```html
   <div style={{ fontSize: '12px', color: 'var(--text-muted)', padding: '8px 0' }}>
     Could not read migration plan
   </div>
   ```

3. **No plan yet** — if plan is null and not pollError: render nothing (empty fragment). The results panel above is sufficient.

4. **Overall progress bar** — when plan exists:
   - Compute `{ complete, total }` via computeProgress(plan)
   - Calculate `pct = total > 0 ? Math.round((complete / total) * 100) : 0`
   - Show text: `{complete}/{total} items ({pct}%)` in fontSize 12px, color var(--text-secondary), marginBottom 6px
   - Progress bar: outer div height 6px, borderRadius 3px, background var(--bg-secondary), overflow hidden. Inner div height 100%, width `${pct}%`, background '#4caf50', borderRadius 3px, transition 'width 0.3s ease'.

5. **Item list** — map over plan.items with index. Separate shared items (type === 'shared') and page items. Render shared items first, then pages. Both use the same row rendering:

   **Collapsed row (clickable for pages with children):**
   ```
   <div onClick={() => toggleExpanded(idx)} style={{ display: 'flex', alignItems: 'center', gap: '6px', padding: '4px 0', cursor: item.children?.length ? 'pointer' : 'default', fontSize: '12px' }}>
     {item.children?.length ? (
       <span style={{ color: 'var(--text-muted)', fontSize: '10px', minWidth: '12px' }}>
         {expanded.has(idx) ? '▼' : '▶'}
       </span>
     ) : (
       <span style={{ color: STATUS_COLOR[item.status], fontSize: '11px', minWidth: '12px' }}>
         {STATUS_SYMBOL[item.status]}
       </span>
     )}
     <span style={{ color: 'var(--text-primary)', flex: 1 }}>{item.name}</span>
     {item.children?.length ? (
       <span style={{ color: 'var(--text-muted)', fontSize: '11px' }}>
         {computePageProgress(item).complete}/{computePageProgress(item).total}
       </span>
     ) : null}
   </div>
   ```

   **Expanded children (when expanded.has(idx) is true):**
   ```
   {item.children?.map((child, ci) => (
     <div key={ci} style={{ display: 'flex', alignItems: 'flex-start', gap: '6px', padding: '2px 0 2px 18px', fontSize: '11px' }}>
       <span style={{ color: STATUS_COLOR[child.status], fontSize: '11px', minWidth: '14px', flexShrink: 0 }}>
         {STATUS_SYMBOL[child.status]}
       </span>
       <div>
         <span style={{ color: 'var(--text-primary)' }}>{child.name}</span>
         {child.notes && (
           <div style={{ color: 'var(--text-muted)', fontSize: '10px', marginTop: '1px' }}>{child.notes}</div>
         )}
       </div>
     </div>
   ))}
   ```

**Status constants (top of file):**
