4. Create `src/zip/extract.ts` with three exported functions:

- `pickZipFile(shell: Shell): Promise<string | null>` — calls `shell.exec('osascript', ['-e', 'POSIX path of (choose file with prompt "Select Webflow export zip" of type {"zip"})'])`. If exit_code !== 0: check stderr for "-128" and return null (user cancelled), otherwise throw. If exit_code === 0: trim stdout, throw if empty, return the path.

- `buildExtractDir(projectPath: string, zipPath: string): string` — extracts the zip filename from the path, strips `.zip` extension (case-insensitive), replaces non-alphanumeric characters (except `-` and `_`) with `-`, truncates to 60 chars, returns `${projectPath}/.shipstudio/tmp/${sanitizedName}`.

- `extractAndVerify(shell: Shell, zipPath: string, extractDir: string, onProgress?: (label: string) => void): Promise<ZipManifest>` —
  (a) calls `shell.exec('unzip', ['-l', zipPath])`, throws on non-zero exit;
  (b) parses manifest via `parseUnzipManifest` (imported from discover.ts);
  (c) calls `shell.exec('mkdir', ['-p', extractDir])`;
  (d) calls onProgress with `Extracting zip... (${manifest.fileCount} files)`;
  (e) calls `shell.exec('unzip', ['-o', zipPath, '-d', extractDir], { timeout: 300000 })`, throws on non-zero exit;
  (f) calls `shell.exec('bash', ['-c', "find '${extractDir}' -type f | wc -l | tr -d ' '"])`;
  (g) parses actual count, throws "Extraction incomplete: expected ~N files, found M. The zip may be corrupted." if actual < expected - 2;
  (h) returns the manifest.

1. Create `src/zip/discover.ts` with two exported functions:

- `parseUnzipManifest(stdout: string): ZipManifest` —
  Parses `unzip -l` output. For each line, use regex `^\s*\d+\s+\d{2}-\d{2}-\d{4}\s+\d{2}:\d{2}\s+(.+)$` to extract the filename portion. Skip header lines (matching `^-{5,}` or `Length\s+Date`), blank lines, and summary lines. Extract total from summary line matching `/(\d+) files/`. If summary regex doesn't match, use entries.length as fileCount. Return `{ fileCount, entries }`.

  CRITICAL: Do NOT use `.split(' ')` to parse rows — filenames with spaces will break. Use the regex column matching pattern.

- `validateWebflowExport(shell: Shell, extractDir: string, entries: string[]): Promise<void>` —
  (a) Check entries for at least one `.html` file at root level: `entries.some(e => e.endsWith('.html') && !e.includes('/'))`. If none found, throw `new Error('No HTML files found — is this a Webflow export?')`.
  (b) Check entries for css/ directory: `entries.some(e => e.startsWith('css/'))`. If none found, throw `new Error('Missing CSS directory — is this a Webflow export?')`.
  (c) Post-extraction check: call `shell.exec('bash', ['-c', "grep -c 'data-wf-site' '" + extractDir + "/index.html' 2>/dev/null || echo 0"])`. Parse stdout as integer. If 0, throw `new Error('No data-wf-site attribute found — this may not be a Webflow export')`.
